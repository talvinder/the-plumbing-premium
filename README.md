# The Plumbing Premium

The systematic tendency for AI infrastructure companies to outperform application-layer companies once a compute paradigm matures — because every layer above the infrastructure eventually commoditizes.

## The Pattern

In every major compute wave, the companies building the plumbing — the infrastructure nobody sees but everyone depends on — end up capturing disproportionate value. Applications compete on features; infrastructure compounds on lock-in. AWS outperformed the web apps it hosted. Stripe outperformed the e-commerce businesses it powered. AI runtime infrastructure is following the same curve.

The premium is not random. It is structural: infrastructure scales with all applications simultaneously, switching costs are high, and margin improves as volume grows.

## Quick Reference

| Paradigm | Plumbing (Infrastructure) | Applications |
|---|---|---|
| Cloud computing | AWS, Azure, GCP | SaaS apps built on top |
| Payments | Stripe, Braintree | E-commerce, fintech apps |
| AI / ML runtime | Modal Labs ($1.1B), Baseten ($2.15B), Fireworks AI ($4B) | AI-powered end applications |
