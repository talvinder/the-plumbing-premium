# The Plumbing Premium

**The systematic outperformance of infrastructure companies over application-layer companies once a compute paradigm matures.**

Every major platform shift produces the same capital allocation mistake: investors flood the application layer first, then discover that infrastructure is where value compounds. AWS outperformed the web apps it hosted. Stripe outperformed the fintech apps it powered. The AI runtime infrastructure layer is following the same curve — and the funding data already confirms it.

This repository contains the framework definition, supporting evidence, and structured data behind The Plumbing Premium.
